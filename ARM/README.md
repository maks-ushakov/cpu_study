# ARM architecture

ARM or Cortex architecture for embeded or smallboard

## Hardware

STM32F103C8T6
----

Board: Blue Pill (USB, UART and GPIO)

Displays:

- 16x2,
- 7segment LED,
- TFT LCD panel on ILI9486/ILI9488L,
- matrix with MAX72

Keyboard:

 - 4x4 Arduino shield
 - custom matrix

## Software

Raspberry PI: Paspbian on QEMU

ARM arch on QEMU

ARM-gcc

ARM-as
